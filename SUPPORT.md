# Support

## Community Support

If you have questions, found a bug, or want to request a feature, please use the [GitHub Issues](https://github.com/connEthics/arazzo-vscode/issues) tracker.

Before opening a new issue, please check if a similar issue has already been reported.

## Commercial Support

For commercial licensing inquiries or dedicated support, please contact the maintainers directly or visit [https://arazzo.connethics.com/](https://arazzo.connethics.com/).
